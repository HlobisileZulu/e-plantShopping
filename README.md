# e-plantShopping

Paradise Nursery is a React shopping app for houseplants built with Vite, React Router, and Redux Toolkit.

## Features
- Landing page with hero image
- Browse plants by category
- Add to cart / disable added button
- Dynamic cart counter
- Increase, decrease, delete cart items
- Total quantity and total price
- Checkout placeholder

## Tech Stack
- React
- Vite
- Redux Toolkit
- React Router DOM

## Install
npm install
npm run dev

## Build
npm run build
